# Contributing to NLP Resources

## How to Contribute
1. Fork the repository
2. Add your resource to the relevant file in /resources
3. Follow the format:
   - **Title of the Blog/Resource**  
  Short one-line description of what this resource covers.  
  Link: https://example.com/blog-post

 Example:
 - **Introduction to NLP with Python**  
  Beginner-friendly blog explaining NLP fundamentals with Python examples.  
  Link: https://towardsdatascience.com/introduction-to-nlp-with-python-xxxx

## Rules
- Must be related to NLP
- No spam or affiliate links
- No duplicate resources
- Resources must be safe and legal

## Review
Maintainers will review your PR and may request changes.
